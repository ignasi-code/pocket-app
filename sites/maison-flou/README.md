# MAISON FLOU Public Site

Static Cloudflare Pages surface for `maisonflou.com`.

Deployment shape:

- framework preset: none

- build output directory: `sites/maison-flou`

Direct upload alternative:

```sh
npx wrangler pages deploy sites/maison-flou --project-name maison-flou
```
